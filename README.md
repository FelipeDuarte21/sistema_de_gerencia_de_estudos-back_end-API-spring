# API RESTFull Sistema de Gerência de Estudo

## Descrição do Projeto
O projeto visa que o usuário consiga ter visão clara e objetiva de como está a sua evolução pessoal dentro de um planejamento de um determinado estudo. Estudo se refere a um curso ou algo que exija tempo a ser concluído.

## Tecnologias Usadas
Em primeiro lugar está API RESTFull trata-se do back-end do sistema. Para o desenvolvimento desta foi usada a linguagem de programação JAVA com auxílio do framework SPRING e dentre os projetos do spring foi utilizado: SPRING BOOT - framework de integração, SPRING WEB - framework principal para construção de API's RestFull, SPRING SECURITY - framework especializado para segurança da aplicação envolvendo a parte de autenticação e autorização com JWT TOKEN, a biblioteca do LOMBOK para deixar as classes modelos mais curtas e mais objetivas e por ultímo e não menos importante o banco de dados POSTGRESQL.
